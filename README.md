# 易宿酒店预订平台

## 项目简介

易宿酒店预订平台是一个现代化的酒店管理系统，包含用户端（移动端）和管理端（PC端）。用户端提供酒店搜索、详情查看、在线预订等功能；管理端提供酒店信息录入、审核管理等功能。项目使用React + TypeScript + Tailwind CSS技术栈，无需后端API，所有数据通过localStorage模拟。

**项目特色**：
- **完整的前后端分离架构**：前端独立运行，后端数据通过localStorage模拟
- **角色权限管理**：商户和管理员角色，不同权限访问不同功能
- **移动端优先设计**：用户端专为移动设备优化，管理端为PC操作优化
- **实时数据同步**：所有操作实时更新本地存储，页面自动刷新
- **现代化UI/UX**：使用Tailwind CSS v4构建美观的用户界面

## 技术栈

- **前端框架**: React 18 + TypeScript
- **样式方案**: Tailwind CSS v4
- **路由管理**: React Router v6
- **状态管理**: Zustand
- **构建工具**: Vite
- **虚拟滚动**: react-window（备用）
- **图标**: Emoji图标 + Unicode字符
- **数据存储**: localStorage（模拟后端API）

## 功能模块

### 用户端（移动端）
1. **首页** (`/user/home`)
   - 轮播Banner展示
   - 酒店搜索表单（城市、日期、关键词、价格、标签筛选）
   - 智能查询功能

2. **酒店列表页** (`/user/list`)
   - 酒店列表展示
   - 筛选和排序功能
   - 分页加载

3. **酒店详情页** (`/user/detail/:id`)
   - 酒店图片轮播（支持触摸滑动）
   - 酒店基础信息展示
   - 房型选择和价格对比
   - 在线预订功能

### 管理端（PC端）
1. **登录/注册页** (`/admin/login`)
   - 双Tab切换（登录/注册）
   - 用户角色选择（商户/管理员）
   - 本地身份验证
   - 预置测试账号

2. **酒店录入页** (`/admin/form`, `/admin/form/:id`)
   - 酒店信息表单（基本信息、标签、房型设置）
   - 房型管理（添加、删除）
   - 提交审核功能
   - 保存草稿功能

3. **审核管理页** (`/admin/audit`)
   - 酒店状态统计（待审核、已发布、已下线、不通过）
   - 酒店列表表格展示
   - 智能操作按钮（根据状态显示不同操作）
   - 审核不通过原因填写

## 项目亮点

### 1. 响应式设计
- 用户端采用移动端优先设计，最大宽度430px，模拟真实移动端体验
- 管理端采用PC端设计，左侧固定导航栏，右侧内容区
- 全面适配不同屏幕尺寸

### 2. 完整的业务逻辑
- 用户端：搜索→列表→详情→预订完整流程
- 管理端：登录→录入→审核闭环流程
- 数据持久化：localStorage模拟后端API，数据持久保存

### 3. 优秀用户体验
- 智能状态管理：不同用户角色显示不同界面和功能
- 实时数据同步：所有操作实时更新localStorage，页面自动刷新
- 表单验证：完整的输入验证和错误提示
- 交互反馈：toast提示、加载状态、确认对话框

## 数据存储

项目使用localStorage存储以下数据：
1. **users** - 用户账户信息（商户/管理员）
2. **currentUser** - 当前登录用户信息
3. **hotels** - 酒店数据（初始化时使用`src/data/hotels.json`）

## 注意事项

1. **本地存储限制**: localStorage有5MB大小限制，生产环境建议使用真实后端API
2. **浏览器兼容性**: 支持现代浏览器（Chrome 90+, Firefox 88+, Safari 14+）
3. **TypeScript严格模式**: 项目使用TypeScript严格模式，确保类型安全
4. **Tailwind CSS v4**: 使用最新版Tailwind CSS，配置简洁
